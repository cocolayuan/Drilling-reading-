针对性跟读清单

把你正在练的三类音从原文里拎出来,集中读。每个词也可以点开听音。

Dark L

词尾的 L · 暗 L

词尾的 dark L(舌后抬高,像含一个 "o")

辅音前的 dark L(old、help、results、involves、animals、always、also)

🔇 看到 L,但不发音

对比 clear L(在元音前,舌尖轻点)

感受:词尾的 L(people、recall、exhale、all)和辅音前的 L(old、help、involves、results)都是 dark——舌后往上压、舌尖可不抵齿龈;clear L(learn、later、language)是舌尖轻点后立刻放开。先慢读 recall → help → all。

清辅音 / 浊辅音

词尾 -s / -ed 的清浊

-s → /z/ · 前面是浊音 → 喉咙振动

-s → /s/ · 前面是清音 → 只送气

-ed → /t/(清音后) · /d/(浊音后)

-ed → /ɪd/ · 前面是 t/d → 多读一个音节

规则:词尾 -s / -ed 跟着前面的音"清浊一致"。手指放喉咙——/z/、/d/ 会震,/s/、/t/ 不震;participants、plants、students、results 因为前面是清辅音,所以是 /ts/;wanted、tested、needed 前是 t/d,要多读一个 /ɪd/。

/ɪ/ vs /aɪ/

短 i 和滑动 i

/ɪ/ 短而松(不是中文"衣")

/aɪ/ 滑动双元音(a → ɪ)

最小对立对 · 读出区别

– – –

注意:/ɪ/ 在这篇里几乎每句都有(it、in、if、this、English、image、breathing、looking、quickly);/aɪ/ 集中在 time、find、while、why、might、trying、identify——刚好对照练。关键:/ɪ/ 短而松、不上滑;/aɪ/ 要从 /a/ 一路滑到 /ɪ/。

Connected speech

连读 · 别一个词一个词蹦

窍门:前一个词的尾辅音/鼻音滑进下一个词的元音(end of an → ˈɛn·də·vən;speaking aloud → ˈspiː·kɪ·ŋə·ˈlaʊd;breathing out → ˈbriː·ðɪ·ŋaʊt),让它们连成一串,不要在每个词后面停。
