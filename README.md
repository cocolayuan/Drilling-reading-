# Drilling Reading · Engoo 逐词音标精读 🇺🇸

每天一篇 **Engoo Daily News(Level 5)**,做成**逐词美式音标**精读页:点任意单词就能听美式发音,用顶部开关只点亮你正在练的那类音(**Dark L**、**清辅音 / 浊辅音**、**/ɪ/ vs /aɪ/**),并配一份针对性跟读清单。所有日子都收录在同一个滚动 10 天日历里——**一个链接,随时回顾,也方便分享给朋友**。
