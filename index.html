针对性跟读清单

把你正在练的三类音从原文里拎出来,集中读。每个词也可以点开听音。

Dark L

词尾的 L · 暗 L

词尾 / 辅音前的 dark L(舌后抬高,像含一个 "o")

🔇 看到 L,但不发音

对比 clear L（在元音前,舌尖轻点）

感受:词尾的 L(goal、told)和辅音前的 L(world、simple)都是 dark——舌后往上压;而 talk / talks 里的 L 拼写有、却完全不发音。clear L(like、slow)是舌尖轻点后立刻放开。先慢读 goal → world → while。

清辅音 / 浊辅音

词尾 -s / -ed 的清浊

-s → /z/ · 前面是浊音 → 喉咙振动

-s → /s/ · 前面是清音 → 只送气

-ed → /d/（浊音后） · /ɪd/（t/d 后多一个音节）

规则:词尾 -s / -ed 跟着前面的音“清浊一致”。手指放喉咙——/z/、/d/ 会震,/s/、/t/ 不震;只有前面是 t/d 时,-ed 才多出一个 /ɪd/ 音节(started)。

/ɪ/ vs /aɪ/

短 i 和滑动 i

/ɪ/ 短而松（不是中文“衣”）

/aɪ/ 滑动双元音（a → ɪ）

最小对立对 · 读出区别

– – –

注意:/ɪ/ 几乎每句都有;这次 /aɪ/ 也很集中——silent、library、time、while…正好成对练。关键:/ɪ/ 短而松、不上滑;/aɪ/ 要从 /a/ 一路滑到 /ɪ/。

连读 · 别一个词一个词蹦

窍门:前一个词的尾辅音滑进下一个词的元音(talk about it → tɔː‿kə‿ˈbaʊ‿tɪt),让它们连成一串,不要在每个词后面停。
